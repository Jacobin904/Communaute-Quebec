## 🚀 Déploiement

Le site est hébergé gratuitement via **GitHub Pages** :

1. Le contenu de la branche `main` (racine du dépôt) est publié automatiquement.
2. Aucune étape de build : c'est du HTML/CSS/JS pur.
3. URL publique : `https://jacobin904.github.io/Communaute-Quebec/`

Pour activer GitHub Pages sur un fork : **Settings → Pages → Source : `main` / `/ (root)` → Save**.

---

## 🔐 Panel administrateur

Une route **`/admin`** donne accès à un tableau de bord (logs de visites, gestion d'accès, liens utiles).

- 🔒 **Accès protégé par mot de passe** (non communiqué publiquement).
- 🎨 Design cohérent avec le site principal.
- 🧩 Prêt à être relié à une base de données ([Supabase](https://supabase.com/)) pour des logs et une blacklist partagés entre tous les appareils.

> ⚠️ **Sécurité :** ne commitez **jamais** de mot de passe, token ou clé API dans ce dépôt public.
